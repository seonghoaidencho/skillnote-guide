# 테마 및 언어

Skillnote는 라이트/다크 테마와 다국어를 지원합니다. 취향에 맞게 설정을 변경할 수 있습니다.

---

## 테마 설정

### 라이트 모드 / 다크 모드

Skillnote는 두 가지 테마를 제공합니다:

- **라이트 모드**: 밝은 배경의 기본 테마
- **다크 모드**: 어두운 배경으로 눈의 피로를 줄여줍니다

### 테마 변경 방법

사이드바 또는 설정 페이지에서 **테마 토글 버튼**을 클릭하면 즉시 전환됩니다.

- 해/달 아이콘 버튼을 클릭하여 전환
- 설정은 브라우저에 저장되어 다음 방문 시에도 유지됩니다

> **Tip:** 야간에는 다크 모드를 사용하면 눈의 피로를 줄이고 집중력을 높일 수 있습니다.

---

## 언어 설정

Skillnote는 **3개 언어**를 지원합니다:

| 언어 | 코드 |
|------|------|
| 한국어 | KR |
| English | EN |
| 中文 | CN |

### 언어 변경 방법

1. 화면 우측 하단의 **언어 토글 버튼**을 클릭합니다
2. 원하는 언어를 선택합니다
3. 모든 UI 텍스트가 즉시 선택한 언어로 전환됩니다

### 언어 설정 저장

- 언어 설정은 브라우저에 저장됩니다
- 로그인한 경우 사용자 프로필에도 동기화되어 다른 기기에서도 동일한 언어가 적용됩니다
- 최초 접속 시 브라우저의 기본 언어를 자동으로 감지합니다

> **Tip:** 영어 학습을 위해 UI 언어를 English로 변경하여 사용해 보는 것도 좋은 방법입니다.

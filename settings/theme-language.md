# 테마 및 언어

Skillnote는 라이트/다크 테마와 다국어 UI를 지원합니다. 취향과 환경에 맞게 설정을 변경해 보세요.

---

## 테마 설정

### 라이트 모드 / 다크 모드

Skillnote는 두 가지 테마를 제공합니다.

| 테마 | 설명 |
|------|------|
| **라이트 모드** | 밝은 배경의 기본 테마 |
| **다크 모드** | 어두운 배경으로 야간이나 저조도 환경에서 눈의 피로를 줄여줍니다 |

### 테마 변경 방법

사이드바에 있는 **태양(sun) 아이콘** 또는 **달(moon) 아이콘** 토글 버튼을 클릭하면 라이트 모드와 다크 모드가 즉시 전환됩니다.

- 현재 **라이트 모드**일 때: 달(moon) 아이콘이 표시됩니다. 클릭하면 다크 모드로 전환됩니다.
- 현재 **다크 모드**일 때: 태양(sun) 아이콘이 표시됩니다. 클릭하면 라이트 모드로 전환됩니다.

### 테마 설정 저장

- 선택한 테마는 **브라우저에 저장**되어 다음 방문 시에도 그대로 유지됩니다.
- 별도의 저장 버튼을 누를 필요 없이 토글하는 즉시 자동 저장됩니다.

> **Tip:** 야간에 오래 학습할 때는 다크 모드를 사용하면 눈의 피로를 줄이고 집중력을 유지하는 데 도움이 됩니다.

## 언어 설정

### 지원 언어

현재 Skillnote가 지원하는 언어는 다음 두 가지입니다.

| 언어 | 표시 이름 |
|------|----------|
| 한국어 | 한국어 (Korean) |
| 영어 | English |

### 언어 변경 방법

설정 페이지에서 언어를 변경할 수 있습니다.

1. 사이드바 하단의 프로필 영역을 클릭합니다.
2. 드롭다운 메뉴에서 **"프로필 설정"** 을 선택합니다.
3. **"계정"** (링크 아이콘) 탭을 클릭합니다.
4. **"언어 설정"** 드롭다운에서 원하는 언어를 선택합니다.

선택 즉시 Skillnote 전체 UI 텍스트가 해당 언어로 전환됩니다.

### 언어 설정 저장

- 언어 설정은 사용자 계정에 저장되어 다른 기기나 브라우저에서 로그인해도 동일한 언어가 적용됩니다.
- 로그인하지 않은 경우에는 브라우저에 저장됩니다.

> **참고:** 현재 지원 언어는 **한국어**와 **English** 두 가지입니다. 향후 추가 언어 지원이 확대될 수 있습니다.

# 프로필 관리

프로필 설정에서는 다른 사용자에게 보이는 개인 정보와 계정 정보를 관리할 수 있습니다.

---

## 설정 페이지 접근

1. 사이드바 하단의 프로필 영역을 클릭합니다.
2. 드롭다운 메뉴에서 **"프로필 설정"** 을 선택합니다.

설정 페이지는 **데스크톱**에서는 왼쪽 사이드바 메뉴와 오른쪽 콘텐츠 영역의 2단 레이아웃으로 표시됩니다. **모바일**에서는 탭을 선택할 때 슬라이드 애니메이션으로 전환됩니다.

---

## 설정 탭 구성

설정 페이지는 4개의 탭으로 구성됩니다.

| 아이콘 | 탭 이름 | 설명 |
|--------|---------|------|
| 사용자(user) 아이콘 | **프로필** | 공개 프로필 정보 관리 |
| 링크(link) 아이콘 | **계정** | 닉네임, 생년월일, 직군 등 개인 정보 |
| 자물쇠(lock) 아이콘 | **비밀번호** | 비밀번호 변경 |
| 신용카드(credit-card) 아이콘 | **결제** | 구독 플랜 확인 및 관리 |

---

## 프로필 탭

### 프로필 이미지

탭 상단에 프로필 아바타가 표시됩니다. 아바타 위에 마우스를 올리면 **"수정"** 오버레이가 나타나며, 클릭하면 이미지를 변경할 수 있습니다.

- 이미지를 등록하지 않으면 블로그 이름의 첫 글자가 기본 아바타로 표시됩니다.
- **Plus** 또는 **Pro** 플랜 사용자라면 아바타 옆에 플랜 배지가 표시됩니다.

### 블로그 이름

- **레이블**: 블로그 이름
- **플레이스홀더**: "블로그 이름을 입력하세요."
- **최대 길이**: 15자
- 블로그 페이지 상단과 게시물 작성자 정보에 표시되는 이름입니다.

### 유저태그

- **레이블**: 유저태그
- 입력 필드 왼쪽에 자물쇠(lock) 아이콘이 표시됩니다.
- 필드를 클릭하거나 마우스를 올리면 **"해당 필드는 수정할 수 없습니다."** 툴팁이 나타납니다.
- 가입 시 설정된 유저태그는 변경할 수 없으며, 최대 20자입니다.

### 소개

- **레이블**: 소개
- **플레이스홀더**: "자신을 소개해주세요."
- **최대 길이**: 200자 (입력창 아래에 `N/200` 형식의 글자 수 카운터가 표시됩니다)
- 블로그 프로필 페이지와 팔로워/팔로잉 등의 소셜 영역에 표시됩니다.

### 소개 링크

- **힌트 텍스트**: "웹사이트, 블로그 또는 소셜 미디어 프로필의 링크를 추가하세요. (3개)"
- 최대 3개의 링크를 등록할 수 있으며, 각 링크에는 전용 아이콘이 표시됩니다.

| 아이콘 | 링크 종류 |
|--------|----------|
| GitHub 아이콘 | GitHub 프로필 URL |
| LinkedIn 아이콘 | LinkedIn 프로필 URL |
| 체인(chain) 아이콘 | 개인 웹사이트 URL |

### 저장 및 취소

- **"취소"** 버튼: 변경 사항을 저장하지 않고 원래 값으로 되돌립니다.
- **"저장"** 버튼: 변경된 정보를 저장합니다.

## 계정 탭

### 닉네임

- **레이블**: 닉네임
- 댓글, 팔로우 알림 등 소셜 상호작용에서 사용되는 이름입니다.

### 생년월일

- **레이블**: 생년월일
- **힌트**: "생년월일을 기준으로 적절한 복습 주기를 설정합니다."
- 날짜 선택 UI(date picker)로 입력합니다.

### 현재 직군

- **레이블**: 현재 직군
- 드롭다운에서 선택합니다. 선택 가능한 직군 목록:

  게임 개발 / 데이터 분석 / 데이터 엔지니어링 / 백엔드 개발 / 정보 보안 / 프론트엔드 개발 / 클라우드 개발 / 프로젝트 매니징 / AI 개발 / AI 연구 (데이터 사이언스) / 마케팅 / 디자인 / 직접 입력

### 학습 분야

- **레이블**: 학습 분야
- **힌트**: "현재 직군과 학습 분야를 기반으로 유사 학습자를 추천합니다."
- 현재 직군과 동일한 드롭다운 목록에서 선택합니다.

### 현재 연차

- **레이블**: 현재 연차
- **힌트**: "연차를 기반으로 적절한 복습 주기를 설정합니다."
- 드롭다운 선택 목록: `0~1년` / `1~3년` / `3~5년` / `5~10년` / `10년 이상`

### 언어 설정

- **레이블**: 언어 설정
- 드롭다운에서 UI 표시 언어를 선택합니다.
- 선택 가능한 언어: **한국어 (Korean)** / **English**

> **참고:** 언어 설정을 변경하면 Skillnote 전체 UI 텍스트가 즉시 해당 언어로 전환됩니다. 자세한 내용은 [테마 및 언어](./theme-language.md) 문서를 참고하세요.

## 비밀번호 탭

현재 비밀번호와 새 비밀번호를 입력한 뒤 **"비밀번호 변경"** 버튼을 클릭하면 비밀번호가 변경됩니다.

- **현재 비밀번호**: 본인 확인을 위해 기존 비밀번호를 입력합니다.
- **새 비밀번호**: 변경할 새 비밀번호를 입력합니다.
- **비밀번호 확인**: 새 비밀번호를 한 번 더 입력하여 일치 여부를 확인합니다.

> **참고:** GitHub 또는 Google 소셜 로그인으로 가입한 계정은 비밀번호 탭이 적용되지 않을 수 있습니다.

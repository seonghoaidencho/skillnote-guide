# 프로필 관리

프로필 설정에서 다른 사용자에게 보이는 개인 정보를 관리할 수 있습니다.

---

## 프로필 설정 접근

1. 사이드바 하단의 프로필 영역을 클릭합니다
2. 드롭다운 메뉴에서 **"프로필 설정"** 을 선택합니다

---

## 기본 정보

### 표시 이름

- 게시물, 댓글, 블로그에 표시되는 이름입니다
- 언제든지 변경할 수 있습니다

### 프로필 이미지

프로필 이미지를 설정하거나 변경하는 방법:

1. 프로필 설정에서 **프로필 이미지** 영역을 클릭합니다
2. **이미지 업로드 모달**이 나타납니다
3. 이미지를 선택하고 원하는 영역으로 **크롭(자르기)** 합니다
4. **"저장"** 을 클릭하면 프로필 이미지가 업데이트됩니다

> **Tip:** 정사각형에 가까운 이미지를 사용하면 프로필 아바타에 잘 표시됩니다.

이미지를 설정하지 않으면 이름의 첫 글자가 기본 아바타로 표시됩니다.

---

## 소셜 링크

프로필에 소셜 미디어 링크를 추가할 수 있습니다:

- **GitHub**: GitHub 프로필 URL
- **Twitter/X**: Twitter 프로필 URL
- **기타 링크**: 개인 웹사이트, 포트폴리오 등

소셜 링크는 블로그 프로필 페이지에 표시됩니다.

---

## 계정 설정

### 이메일

- 계정에 등록된 이메일 주소를 확인할 수 있습니다
- Google 로그인 사용자의 경우 Google 계정 이메일이 표시됩니다

### 로그아웃

1. 사이드바 하단의 프로필 영역을 클릭합니다
2. 드롭다운 메뉴에서 **"로그아웃"** 을 선택합니다

> **주의:** 공용 컴퓨터에서 사용 후에는 반드시 로그아웃해 주세요.

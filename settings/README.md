# 설정

Skillnote의 다양한 설정을 통해 자신에게 맞는 환경을 구성할 수 있습니다.

---

## 이 섹션에서 다루는 내용

| 항목 | 설명 |
|------|------|
| [프로필 관리](profile.md) | 표시 이름, 프로필 이미지, 소셜 링크 관리 |
| [테마 및 언어](theme-language.md) | 라이트/다크 테마와 언어 설정 |
| [구독 플랜](subscription.md) | Free, Plus, Pro 플랜 안내 |

---

## 설정 접근 방법

사이드바 하단의 **프로필 영역**을 클릭하면 드롭다운 메뉴가 나타납니다. 여기서 **"프로필 설정"** 을 선택하면 설정 페이지로 이동합니다.

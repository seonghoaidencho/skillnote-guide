# 작성하기

Skillnote의 핵심은 **배운 것을 기록하고, AI가 그것을 복습 문제로 만들어주는 것**입니다. 직접 게시물을 작성하면, AI가 내용을 분석해 스페이스드 리피티션 방식의 퀴즈를 자동으로 생성합니다.

## 작성 기능 개요

- **게시물 작성**: 강력한 리치 텍스트 에디터로 학습 내용을 자유롭게 기록합니다.
- **PDF 업로드**: PDF 문서를 업로드하면 AI가 요약해 학습 게시물로 자동 변환합니다.
- **게시물 관리**: 작성한 게시물과 임시저장 글을 한곳에서 확인하고 관리합니다.

## 작동 원리

```
게시물 작성 → 게시 → AI 분석 → 복습 문제 자동 생성 → 스페이스드 리피티션 복습
```

게시물을 발행하는 순간 AI가 내용을 읽고 핵심 개념을 파악해 퀴즈를 만듭니다. 따로 문제를 만들 필요 없이, 잘 쓴 게시물 하나가 완성된 복습 세트가 됩니다.

# 시리즈 관리

시리즈는 관련 게시물을 하나의 묶음으로 정리하는 기능입니다. 주제별로 학습 내용을 체계적으로 구성하고, 블로그 프로필에서 방문자에게 일목요연하게 보여줄 수 있습니다.

## 시리즈란?

예를 들어 "Python 기초", "알고리즘 스터디", "React 학습기"처럼 여러 게시물이 하나의 학습 흐름을 이루는 경우, 시리즈로 묶으면 관련 게시물을 한눈에 파악할 수 있습니다.

- 시리즈에 속한 게시물은 블로그 프로필에서 시리즈 단위로 묶여 표시됩니다.
- 시리즈 내 게시물은 순서대로 정렬되어 학습 흐름을 명확하게 보여줍니다.

## 새 시리즈 만들기

에디터의 오른쪽 사이드바에서 시리즈를 생성할 수 있습니다.

1. 에디터에서 게시물 작성 중 오른쪽 사이드바의 **시리즈** 섹션을 찾습니다.
2. **새 시리즈 만들기**를 선택합니다.
3. 시리즈 이름을 입력하고 저장합니다.
4. 새로 만든 시리즈가 현재 게시물에 자동으로 지정됩니다.

## 게시물에 시리즈 지정하기

### 작성 중 지정

게시물 작성 또는 편집 시 에디터 오른쪽 사이드바의 **시리즈** 드롭다운에서 원하는 시리즈를 선택합니다.

1. 에디터 오른쪽 사이드바에서 **시리즈** 섹션을 찾습니다.
2. 드롭다운에서 기존 시리즈를 선택하거나 새 시리즈를 만듭니다.
3. 게시물을 저장하거나 발행하면 시리즈 지정이 함께 저장됩니다.

> **Tip:** 시리즈는 게시 후에도 에디터에서 수정할 수 있습니다. 기존 게시물을 열어 사이드바에서 시리즈를 변경하면 됩니다.

## 블로그 프로필에서 시리즈 표시

시리즈에 게시물이 하나 이상 포함되면, 내 블로그 프로필 페이지에 시리즈가 표시됩니다. 방문자는 시리즈를 클릭해 해당 시리즈의 모든 게시물을 순서대로 볼 수 있습니다.

> **Tip:** 시리즈 이름을 명확하게 정하면 방문자가 관심 있는 학습 주제를 쉽게 찾을 수 있습니다. 예를 들어 "JavaScript 기초 다지기"처럼 구체적인 이름이 "공부 기록"보다 효과적입니다.

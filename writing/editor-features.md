# 에디터 기능 가이드

Skillnote의 에디터는 TipTap 기반의 블록 에디터입니다. 마크다운 단축키, 슬래시 명령어, 드래그 앤 드롭 등 다양한 방식으로 콘텐츠를 구성할 수 있습니다.

## 텍스트 서식

### 제목

툴바 또는 마크다운 문법으로 H1부터 H6까지 제목을 지정할 수 있습니다.

```
# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
```

### 기본 서식

| 서식 | 단축키 | 마크다운 |
|---|---|---|
| **굵게** | `Ctrl/Cmd + B` | `**텍스트**` |
| *기울임* | `Ctrl/Cmd + I` | `*텍스트*` |
| <u>밑줄</u> | `Ctrl/Cmd + U` | — |
| ~~취소선~~ | — | `~~텍스트~~` |

## 목록

### 순서 없는 목록

툴바의 목록 아이콘을 클릭하거나 `-` 또는 `*`를 입력하고 스페이스를 누릅니다.

- 항목 1
- 항목 2
  - 중첩 항목

### 순서 있는 목록

숫자 `1.`을 입력하고 스페이스를 누릅니다.

1. 첫 번째
2. 두 번째
3. 세 번째

### 체크리스트 (할 일 목록)

`[ ]`을 입력하고 스페이스를 누르면 체크박스 항목이 생성됩니다.

- [ ] 완료되지 않은 항목
- [x] 완료된 항목

## 표

슬래시 명령어(`/표` 또는 `/table`)로 표를 삽입합니다. 셀을 클릭하면 행과 열 추가, 삭제 옵션이 나타납니다.

> **Tip:** 표 안에서 `Tab` 키를 누르면 다음 셀로 이동합니다. 마지막 셀에서 `Tab`을 누르면 새 행이 추가됩니다.

## 코드 블록

백틱 세 개(` ``` `)를 입력하고 언어 이름을 입력하면 코드 블록이 생성됩니다. highlight.js/lowlight 기반의 문법 강조가 적용됩니다.

````
```python
def hello():
    print("Hello, Skillnote!")
```
````

지원 언어 예시: `python`, `javascript`, `typescript`, `java`, `go`, `rust`, `sql`, `bash`, `html`, `css` 등 주요 언어를 모두 지원합니다.

인라인 코드는 백틱 하나(`` ` ``)로 감쌉니다: `` `변수명` ``

## 이미지

이미지를 추가하는 방법은 두 가지입니다.

- **드래그 앤 드롭**: 이미지 파일을 에디터 위로 끌어다 놓습니다.
- **붙여넣기**: 클립보드의 이미지를 `Ctrl/Cmd + V`로 바로 붙여넣습니다.

업로드된 이미지는 AWS S3에 안전하게 저장됩니다.

> **주의:** 지원하는 이미지 형식은 JPG, PNG, GIF, WebP입니다. 파일 크기 제한은 플랜에 따라 다를 수 있습니다.

## 수학 수식

KaTeX를 이용해 수학 수식을 작성할 수 있습니다.

- **인라인 수식**: `$수식$` 형태로 작성합니다. 예: `$E = mc^2$`
- **블록 수식**: 슬래시 명령어(`/수식` 또는 `/math`)로 블록 수식을 삽입합니다.

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

## YouTube 동영상 삽입

YouTube URL을 에디터에 붙여넣으면 자동으로 동영상이 임베드됩니다. 슬래시 명령어(`/youtube`)를 사용할 수도 있습니다.

> **Tip:** YouTube 단축 URL(`youtu.be/...`)과 전체 URL(`youtube.com/watch?v=...`) 모두 지원합니다.

## 이모지

슬래시 명령어(`/이모지` 또는 `/emoji`)를 입력하거나 `:` 뒤에 이모지 이름을 입력하면 이모지 선택기가 열립니다.

## 블록 재정렬 (드래그 앤 드롭)

각 콘텐츠 블록 왼쪽에 마우스를 올리면 드래그 핸들(⠿)이 나타납니다. 핸들을 잡고 원하는 위치로 끌어다 놓으면 블록 순서를 변경할 수 있습니다.

## 슬래시 명령어

본문에서 `/`를 입력하면 빠른 삽입 메뉴가 열립니다. 키워드를 입력해 원하는 블록을 빠르게 찾아 삽입할 수 있습니다.

| 명령어 | 삽입되는 블록 |
|---|---|
| `/h1` ~ `/h6` | 제목 1~6 |
| `/목록`, `/ul` | 순서 없는 목록 |
| `/번호`, `/ol` | 순서 있는 목록 |
| `/체크`, `/todo` | 체크리스트 |
| `/표`, `/table` | 표 |
| `/코드`, `/code` | 코드 블록 |
| `/수식`, `/math` | 수학 수식 블록 |
| `/이미지`, `/image` | 이미지 업로드 |
| `/youtube` | YouTube 동영상 삽입 |
| `/이모지`, `/emoji` | 이모지 선택기 |

## 키보드 단축키 모음

| 단축키 | 기능 |
|---|---|
| `Ctrl/Cmd + B` | 굵게 |
| `Ctrl/Cmd + I` | 기울임 |
| `Ctrl/Cmd + U` | 밑줄 |
| `Ctrl/Cmd + Z` | 실행 취소 |
| `Ctrl/Cmd + Shift + Z` | 다시 실행 |
| `Ctrl/Cmd + A` | 전체 선택 |
| `Tab` | 들여쓰기 (목록, 표 내부) |
| `Shift + Tab` | 내어쓰기 |
| `Enter` | 새 단락 또는 새 항목 |
| `Shift + Enter` | 단락 내 줄바꿈 |

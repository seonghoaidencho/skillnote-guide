# 게시물 관리

작성한 학습 기록을 관리하세요.

내 게시물 페이지(`/posts`)에서 발행된 게시물과 임시저장 게시물을 모두 확인하고 관리할 수 있습니다.

---

## 페이지 구성

페이지 상단에는 제목 **게시물**과 함께 부제목 **작성한 학습 기록을 관리하세요**가 표시됩니다. 오른쪽 상단의 **+ 새 게시물** 버튼을 클릭하면 새 게시물 에디터로 이동합니다.

### 탭

게시물은 두 탭으로 구분됩니다.

| 탭 | 아이콘 | 설명 |
|---|---|---|
| **게시됨** | 문서 아이콘 + 게시물 수 뱃지 | 발행된 게시물 목록 |
| **임시저장됨** | 시계 아이콘 + 임시저장 수 뱃지 | 아직 발행하지 않은 임시저장 목록 |

---

## 검색

페이지 상단 검색창의 플레이스홀더 텍스트는 **제목, 내용, 태그 검색**입니다. 검색어를 입력하면 제목·내용·태그를 기준으로 목록이 실시간 필터링됩니다.

---

## 필터 및 정렬

### 공개 상태 필터 (게시됨 탭 전용)

| 옵션 | 설명 |
|---|---|
| **전체** | 공개·비공개 게시물 모두 표시 |
| **전체 공개** | 전체 공개 게시물만 표시 |
| **나만 보기** | 나만 보기 게시물만 표시 |

### 정렬

드롭다운에서 정렬 순서를 선택합니다.

| 옵션 | 설명 |
|---|---|
| **최신 순** | 최근에 작성된 순서로 표시 |
| **오래된 순** | 오래된 순서로 표시 |
| **최근 업데이트 순** | 최근에 수정된 순서로 표시 |

### 태그 필터

깔때기(funnel) 아이콘을 클릭하면 태그 필터 패널이 열립니다.

- 검색 입력란으로 태그를 검색할 수 있습니다.
- **모든 태그** 항목을 선택하면 태그 필터를 초기화합니다.
- 개별 태그를 선택해 해당 태그가 달린 게시물만 필터링합니다.

---

## 게시물 목록 항목

각 게시물 항목에는 다음 정보가 표시됩니다.

- **체크박스**: 일괄 작업을 위해 게시물을 선택합니다.
- **제목**: 클릭하면 해당 게시물 페이지로 이동합니다.
- **미리보기 텍스트**: 본문 일부가 요약되어 표시됩니다.
- **태그**: 최대 3개까지 표시되며, 초과하는 경우 `...`으로 표시됩니다.
- **날짜**: 작성 또는 발행 날짜
- **조회 수 / 좋아요 수**: **조회 N · 좋아요 N** 형식으로 표시됩니다.
- **공개 상태 아이콘** (게시됨 탭): 지구본 아이콘(전체 공개) 또는 자물쇠 아이콘(나만 보기)
- **이어 쓰기 버튼** (임시저장됨 탭): 해당 임시저장 게시물의 에디터로 이동합니다.
- **더보기 메뉴 (`...`)**: 추가 작업 메뉴를 엽니다.

### 더보기 메뉴 항목

| 항목 | 조건 | 설명 |
|---|---|---|
| **통계** | 게시됨 탭 전용 | 해당 게시물의 조회·좋아요 통계를 확인합니다. |
| **수정** | 항상 표시 | 에디터를 열어 게시물을 수정합니다. |
| **삭제** | 항상 표시 | 게시물을 삭제합니다. |

---

## 일괄 작업

게시물 항목의 체크박스를 선택하면 목록 상단에 일괄 작업 버튼이 나타납니다.

### 공개 상태 일괄 변경 (게시됨 탭 전용)

**공개 상태 변경 (N)** 드롭다운을 클릭하면 다음 옵션이 표시됩니다.

| 옵션 | 아이콘 | 설명 |
|---|---|---|
| **전체 공개로 설정** | 지구본 | 선택한 게시물을 전체 공개로 변경 |
| **나만 보기로 설정** | 자물쇠 | 선택한 게시물을 나만 보기로 변경 |

괄호 안의 숫자(N)는 현재 선택된 게시물 수를 나타냅니다.

### 일괄 삭제

**선택 삭제 (N)** 버튼(휴지통 아이콘)을 클릭하면 선택한 게시물이 삭제됩니다.

> **주의:** 삭제된 게시물과 해당 게시물에서 생성된 복습 문제는 복구할 수 없습니다. 삭제 전에 반드시 확인하세요.

---

## 빈 상태 안내

게시물이 없거나 검색 결과가 없는 경우 다음 안내 메시지가 표시됩니다.

### 검색 결과 없음

> **검색 결과가 없어요**
> 다른 검색어로 시도해보세요

### 게시물 없음 (검색 없이)

> **게시물이 없습니다.**
> 첫 번째 학습 기록을 작성해보세요

**새 게시물** 버튼이 함께 표시되어 바로 작성을 시작할 수 있습니다.

# 게시물 관리

내 게시물 페이지(`/posts`)에서 작성한 모든 게시물과 임시저장 글을 확인하고 관리할 수 있습니다.

## 내 게시물 페이지

사이드바에서 **내 게시물**을 클릭하면 `/posts` 페이지로 이동합니다. 이 페이지에서 발행된 게시물과 임시저장 게시물을 탭으로 구분해 확인할 수 있습니다.

### 탭 구성

| 탭 | 내용 |
|---|---|
| **게시물** | 발행된 모든 게시물 목록 |
| **임시저장** | 아직 발행하지 않은 임시저장 게시물 목록 |

## 게시물 카드 정보

각 게시물 카드에는 다음 정보가 표시됩니다.

- **제목**: 게시물 제목
- **태그**: 게시물에 지정된 태그 목록
- **작성일**: 게시물 작성 또는 발행 날짜
- **조회 수**: 해당 게시물이 조회된 횟수
- **문제 수**: AI가 생성한 복습 문제 수

## 검색

페이지 상단의 검색창에서 내 게시물을 검색할 수 있습니다. 제목 기준으로 검색 결과가 필터링됩니다.

> **Tip:** 태그를 잘 관리하면 검색 없이도 원하는 게시물을 빠르게 찾을 수 있습니다.

## 게시물 편집

게시물 카드를 클릭하면 에디터가 열려 내용을 수정할 수 있습니다. 수정 후 다시 **게시하기**를 클릭하면 변경 내용이 저장되고 AI가 문제를 재생성합니다.

> **주의:** 게시물 내용을 크게 수정하면 기존에 생성된 복습 문제가 변경될 수 있습니다.

## 게시물 삭제

### 개별 삭제

게시물 카드의 삭제 버튼(휴지통 아이콘)을 클릭해 게시물을 삭제합니다. 삭제 전 확인 다이얼로그가 표시됩니다.

### 일괄 삭제

여러 게시물을 한 번에 삭제하려면 일괄 삭제 기능을 사용합니다.

1. 삭제할 게시물의 체크박스를 선택합니다.
2. 여러 게시물을 선택할 수 있습니다.
3. **선택 삭제** 버튼을 클릭합니다.
4. 확인 다이얼로그에서 삭제를 확정합니다.

> **주의:** 삭제된 게시물과 해당 게시물에서 생성된 복습 문제는 복구할 수 없습니다. 삭제 전에 반드시 확인하세요.

# PDF 업로드

PDF 문서를 업로드하면 AI가 내용을 요약해 학습 게시물로 자동 변환합니다. 강의 자료, 논문, 교재 등을 빠르게 Skillnote 학습 콘텐츠로 만들 수 있습니다.

## 사용 방법

### PDF 선택

두 가지 방법으로 PDF를 업로드할 수 있습니다.

- **드래그 앤 드롭**: 홈 화면의 업로드 영역으로 PDF 파일을 끌어다 놓습니다.
- **파일 선택**: 홈 화면의 **PDF 업로드** 버튼을 클릭하고 파일을 선택합니다.

여러 파일을 한 번에 선택해 동시에 업로드할 수 있습니다.

### 처리 과정

PDF를 선택하면 다음 단계가 자동으로 진행됩니다.

```
PDF 선택 → 유효성 검사 (페이지 수, 파일 형식) → 클라이언트에서 내용 파싱 → AI 요약 → 임시저장 게시물로 자동 생성
```

1. **유효성 검사**: 플랜별 페이지 수 제한을 확인합니다.
2. **파싱**: 브라우저에서 PDF 텍스트를 추출합니다.
3. **AI 요약**: AI가 핵심 내용을 분석하고 구조화된 학습 게시물로 요약합니다.
4. **임시저장 생성**: 요약된 내용이 임시저장 게시물로 저장됩니다. 내 게시물 페이지의 **임시저장** 탭에서 확인할 수 있습니다.

## 플랜별 페이지 제한

| 플랜 | 업로드 가능 페이지 수 |
|---|---|
| **Free** | 제한된 페이지 수 |
| **Plus** | 더 높은 페이지 제한 |
| **Pro** | 최대 페이지 제한 |

> **Tip:** 플랜별 정확한 페이지 수 제한은 설정 페이지의 플랜 정보에서 확인할 수 있습니다.

## 진행 상황 확인

여러 PDF를 동시에 업로드하는 경우, 화면 하단에 **진행 상황 오버레이**가 표시됩니다. 각 파일의 처리 단계와 완료 여부를 실시간으로 확인할 수 있습니다.

- 업로드 중인 파일 목록과 각각의 진행 상태가 표시됩니다.
- 처리가 완료된 파일은 체크 표시로 확인됩니다.
- 오류가 발생한 경우 해당 파일에 오류 메시지가 표시됩니다.

## 중단 후 복구

업로드 도중 페이지를 새로고침하거나 브라우저를 닫은 경우, 다음 방문 시 **복구 다이얼로그**가 자동으로 나타납니다. 중단된 업로드를 이어서 진행할지 선택할 수 있습니다.

> **주의:** 복구 다이얼로그를 무시하고 닫으면 해당 업로드 세션의 진행 정보가 삭제됩니다. 중요한 파일은 다시 업로드해야 합니다.

## 좋은 결과를 위한 팁

AI 요약의 품질은 PDF의 특성에 따라 달라질 수 있습니다. 더 나은 결과를 위해 아래 사항을 참고하세요.

- **텍스트 기반 PDF를 사용하세요.** 스캔한 이미지 PDF는 텍스트 추출이 어려워 요약 품질이 낮을 수 있습니다.
- **명확한 구조의 문서가 좋습니다.** 제목, 소제목, 단락이 잘 구분된 문서일수록 AI가 핵심을 더 잘 파악합니다.
- **핵심 내용이 집중된 문서를 선택하세요.** 광고나 불필요한 내용이 많은 PDF보다 학습 내용 위주의 문서가 효과적입니다.
- **생성된 임시저장 게시물을 검토하세요.** AI 요약은 자동 생성이므로, 게시 전에 내용을 직접 확인하고 필요한 경우 수정하는 것을 권장합니다.

> **Tip:** 임시저장으로 생성된 게시물을 에디터에서 열어 내용을 보완하면 AI가 더 정확하고 풍부한 복습 문제를 생성합니다.

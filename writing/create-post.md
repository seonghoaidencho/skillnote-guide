# 새 게시물 작성하기

Skillnote에서 학습 게시물을 작성하는 방법을 안내합니다. 게시물을 발행하면 AI가 자동으로 내용을 분석해 복습 문제를 생성합니다.

## 에디터 열기

새 게시물 작성을 시작하는 방법은 두 가지입니다.

- **사이드바**: 왼쪽 사이드바의 **새 학습 작성** 버튼 클릭
- **홈 빠른 시작**: 홈 화면의 빠른 시작 영역에서 **새 학습 작성** 선택

에디터는 `/newpost` 페이지에서 열리며, TipTap 기반의 깔끔한 리치 텍스트 편집 환경을 제공합니다.

## 에디터 화면 구성

에디터는 크게 두 영역으로 나뉩니다.

- **편집 영역**: 제목과 본문을 자유롭게 작성하는 공간입니다. 상단 툴바에서 서식을 적용할 수 있습니다.
- **오른쪽 사이드바**: 게시물의 공개 설정, 태그, 시리즈를 관리하는 옵션 패널입니다.

## 작성 흐름

### 1단계: 내용 작성

제목 입력란에 게시물 제목을 입력하고, 본문 영역에 학습한 내용을 작성합니다. 툴바를 활용해 제목, 굵기, 코드 블록 등 다양한 서식을 적용할 수 있습니다. 에디터의 상세 기능은 [에디터 기능 가이드](editor-features.md)를 참고하세요.

### 2단계: 태그 설정

오른쪽 사이드바의 태그 영역에서 게시물에 어울리는 태그를 추가합니다. 태그는 나중에 게시물을 검색하고 분류하는 데 도움이 됩니다.

### 3단계: 공개 설정

사이드바의 공개 설정 토글에서 게시물 공개 범위를 선택합니다.

| 설정 | 설명 |
|---|---|
| **공개** | 내 블로그 프로필에서 누구나 게시물을 볼 수 있습니다. |
| **비공개** | 나만 볼 수 있습니다. |

### 4단계: 시리즈 지정 (선택)

관련 게시물을 묶고 싶다면 사이드바에서 시리즈를 선택합니다. 자세한 내용은 [시리즈 관리](series.md)를 참고하세요.

### 5단계: 게시 또는 임시저장

- **게시하기**: 작성을 마치면 오른쪽 상단의 **게시하기** 버튼을 클릭합니다. 게시 즉시 AI가 내용을 분석해 복습 문제를 자동으로 생성합니다.
- **임시저장**: 아직 완성되지 않았다면 **임시저장** 버튼으로 저장해 두고 나중에 이어서 작성할 수 있습니다.

> **Tip:** 에디터는 작성 중 자동저장 기능을 제공합니다. 실수로 탭을 닫더라도 임시저장본이 남아 있으니 안심하세요.

## 게시 후 AI 문제 생성

게시물을 발행하면 Skillnote AI가 다음 과정을 자동으로 수행합니다.

1. 게시물 내용 분석
2. 핵심 개념 및 사실 추출
3. 스페이스드 리피티션용 복습 문제 생성

생성된 문제는 **복습** 탭에서 바로 확인하고 풀 수 있습니다.

## 좋은 복습 문제를 만드는 작성 팁

AI가 더 정확하고 유용한 문제를 생성할 수 있도록 아래 방법을 참고하세요.

- **핵심 개념을 명확하게 서술합니다.** "A는 B이다" 형태의 문장이 좋은 문제로 이어집니다.
- **구체적인 예시를 포함합니다.** 추상적인 설명보다 구체적인 예시가 이해와 문제 생성에 모두 도움이 됩니다.
- **적당한 분량을 유지합니다.** 너무 짧으면 문제가 생성되지 않을 수 있고, 너무 길면 핵심이 분산될 수 있습니다.
- **주제를 하나로 집중합니다.** 한 게시물에 하나의 주제를 깊게 다루는 것이 여러 주제를 얕게 다루는 것보다 효과적입니다.
- **제목을 구체적으로 작성합니다.** 명확한 제목이 AI의 내용 이해를 돕습니다.

> **Tip:** 오늘 배운 것을 자신의 언어로 설명하듯 작성하면 AI가 더 좋은 문제를 만들어냅니다.

# 화면 구성

Skillnote의 전체 화면 구조와 각 영역의 역할을 설명합니다. 사이드바, 홈 대시보드, 게시물 읽기 레이아웃을 차례로 살펴보세요.

---

## 전체 레이아웃

Skillnote의 기본 화면은 왼쪽 사이드바와 오른쪽 메인 콘텐츠 영역으로 나뉩니다.

```
┌──────────────┬──────────────────────────────────────┐
│              │                                      │
│  왼쪽         │  메인 콘텐츠 영역                      │
│  사이드바      │  (현재 페이지에 따라 변경)               │
│              │                                      │
└──────────────┴──────────────────────────────────────┘
```

사이드바는 접기/펼치기가 가능합니다. 접힌 상태(레일 모드)에서는 아이콘만 표시되고, 펼친 상태에서는 아이콘과 메뉴 이름이 함께 표시됩니다.

---

## 왼쪽 사이드바

### 새 학습 게시물 작성

사이드바 최상단에는 펜 아이콘이 있는 **"새 학습 게시물 작성"** 버튼이 있습니다. 클릭하면 `/posts/newpost` 에디터 페이지로 이동합니다. 로그인이 필요하며, 로그인하지 않은 상태에서 클릭하면 로그인 모달이 나타납니다.

### 탐색하기 그룹

| 메뉴 | 아이콘 | 이동 경로 | 설명 |
|------|--------|-----------|------|
| 홈 | 홈 아이콘 | `/` | 오늘의 복습과 빠른 시작이 있는 대시보드 |
| 탐색 | 탐색 아이콘 | `/explore` | 다른 사용자의 공개 게시물 탐색 |

### 기록하기 그룹

| 메뉴 | 아이콘 | 이동 경로 | 설명 |
|------|--------|-----------|------|
| 게시물 | 문서 아이콘 | `/posts` | 내가 작성한 학습 게시물 목록 |

### 학습하기 그룹

| 메뉴 | 아이콘 | 이동 경로 | 설명 |
|------|--------|-----------|------|
| 오늘의 복습 | 댓글 아이콘 | `/questions` | AI가 생성한 오늘의 복습 질문 |

### 내 블로그

사이드바 하단에 **"내 블로그"** 링크가 있습니다. 클릭하면 내 공개 블로그(`/{유저태그}`)로 이동합니다.

### 프로필 드롭다운

사이드바 맨 아래에는 내 프로필 정보가 표시됩니다. 클릭하면 드롭다운 메뉴가 열리며 다음 정보와 버튼을 확인할 수 있습니다.

- 닉네임, 유저태그, 이메일 주소, 프로필 아바타
- **설정** 버튼: 계정 및 블로그 설정 페이지로 이동
- **로그아웃** 버튼: 현재 세션을 종료하고 로그아웃

### 사이드바 접기/펼치기

사이드바 오른쪽 가장자리의 토글 버튼을 클릭하면 사이드바를 접거나 펼칠 수 있습니다.

- **펼침 상태**: 아이콘 + 메뉴 이름 표시
- **접힘 상태 (레일 모드)**: 아이콘만 표시되어 메인 콘텐츠 영역이 넓어집니다

## 홈 대시보드

로그인 후 **"홈"** 메뉴를 클릭하거나 `/` 경로에 접근하면 홈 대시보드가 표시됩니다.

### 날짜 및 인사말

화면 상단에 오늘 날짜가 표시됩니다 (예: **"2월 27일 목요일"**).

그 아래에는 접속 시간대에 따라 다른 인사말이 랜덤으로 표시됩니다. 시간대별로 4가지 문구 중 하나가 무작위로 선택됩니다.

- **오전**: 예) **"좋은 아침입니다. 오늘도 한 걸음만 같이 가볼까요?"**
- **오후**: 오후에 어울리는 격려 문구
- **저녁 이후**: 저녁 시간에 어울리는 인사 문구

### 오늘의 복습 섹션

오늘 복습할 질문이 있는지 여부에 따라 두 가지 상태 중 하나가 표시됩니다.

**복습 질문이 있을 때**

- 책 펼침 아이콘과 함께 복습할 질문 수 (**"N개 문제"**) 와 관련 학습 자료 수 (**"N개 학습 자료"**) 가 표시됩니다.
- **"시작하기"** 버튼을 클릭하면 `/questions` 복습 페이지로 바로 이동합니다.

**복습 질문이 없을 때 (오늘 복습 완료)**

- 하트 아이콘과 함께 **"오늘 복습 완료!"** 메시지가 표시됩니다.
- **"내일 새로운 질문이 준비됩니다"** 안내 문구로 내일 다시 복습할 수 있음을 알려줍니다.

### 빠른 시작 섹션

자주 사용하는 기능으로 바로 이동할 수 있는 2개의 카드가 나란히 표시됩니다.

| 카드 | 부제목 | 아이콘 | 색상 | 기능 |
|------|--------|--------|------|------|
| **글쓰기** | 새 기록 작성 | 펜 아이콘 | 파란색 | 새 게시물 에디터로 이동 |
| **PDF 업로드** | PDF로 초안 생성하기 | 업로드 아이콘 | 보라색 | PDF를 업로드해 AI 초안 생성 |

## 게시물 읽기 레이아웃

게시물을 열면 화면이 두 영역으로 나뉩니다.

```
┌──────────────────────────┬──────────────────────┐
│                          │  ┌──────────────────┐ │
│  메인 콘텐츠               │  │  정보 │ 댓글 │ 복습 │ │
│  (게시물 본문)              │  └──────────────────┘ │
│                          │                      │
│                          │  탭 내용 표시 영역       │
│                          │                      │
└──────────────────────────┴──────────────────────┘
```

### 메인 콘텐츠 (왼쪽)

게시물의 제목, 본문, 태그 등 주요 내용이 표시됩니다.

### 댓글 사이드바 (오른쪽)

오른쪽 사이드바에는 3개의 탭이 있습니다.

| 탭 | 내용 |
|----|------|
| **정보** | 게시물의 메타 정보 (작성일, 태그, 시리즈 등) |
| **댓글** | 이 게시물에 달린 댓글 목록과 댓글 작성 폼 |
| **복습** | 이 게시물을 바탕으로 AI가 생성한 복습 질문 |

원하는 탭을 클릭해 게시물 본문을 읽으면서 댓글을 확인하거나 복습 질문을 바로 풀어볼 수 있습니다.

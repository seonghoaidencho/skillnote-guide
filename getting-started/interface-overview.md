# 화면 구성

Skillnote의 주요 화면 구조와 각 영역의 역할을 설명합니다.

---

## 전체 레이아웃

Skillnote의 기본 화면은 세 가지 영역으로 구성됩니다.

```
┌─────────────────────────────────────────────────┐
│  상단 헤더 (사이드바 토글, 페이지 제목)              │
├──────────────┬──────────────────────────────────┤
│              │                                  │
│  왼쪽         │  메인 콘텐츠 영역                  │
│  사이드바      │  (현재 페이지에 따라 변경)           │
│              │                                  │
│              │                                  │
└──────────────┴──────────────────────────────────┘
```

---

## 왼쪽 사이드바

사이드바는 Skillnote의 주요 페이지로 이동하는 내비게이션 메뉴입니다.

| 메뉴 항목 | 설명 |
|-----------|------|
| 홈 | 오늘의 복습 카드와 빠른 시작 섹션이 있는 대시보드 |
| 탐색 | 다른 사용자가 공개한 학습 게시물 탐색 |
| 내 게시물 | 내가 작성한 학습 게시물 목록 |
| 복습 | AI가 생성한 질문으로 학습 내용 복습 |
| 프로필 설정 | 표시 이름, 프로필 이미지 등 계정 정보 관리 |

---

## 상단 헤더

- **사이드바 토글** — 사이드바를 접거나 펼칩니다. 화면이 좁을 때 콘텐츠 영역을 넓게 사용할 수 있습니다.
- **페이지 제목** — 현재 보고 있는 페이지의 이름을 표시합니다.

---

## 홈 대시보드

로그인 후 처음 만나는 화면입니다.

### 시간대별 인사말

접속 시간에 따라 다른 인사말이 표시됩니다.

- 오전: **"좋은 아침이에요"**
- 오후: **"좋은 오후에요"**
- 저녁 이후: 저녁 인사

### 오늘의 복습 카드

- 오늘 복습할 **질문 수**와 해당 질문이 포함된 **게시물 수**를 표시합니다.
- **"복습 시작"** 버튼을 클릭하면 바로 복습을 시작할 수 있습니다.

> **Tip:** 매일 꾸준히 복습하면 AI가 학습 패턴을 분석해 더 효과적인 질문을 제공합니다.

### 빠른 시작 섹션

홈에서 자주 사용하는 기능으로 바로 이동할 수 있습니다.

| 버튼 | 기능 |
|------|------|
| 새 학습 작성 | 에디터를 열고 새 학습 게시물을 작성합니다. |
| PDF 업로드 | PDF 파일을 업로드해 AI 요약을 받습니다. |

---

## 게시물 읽기 레이아웃 (PostLayout)

게시물을 열면 화면이 두 영역으로 나뉩니다.

```
┌───────────────────────────┬─────────────────────┐
│                           │                     │
│  메인 콘텐츠               │  댓글 사이드바        │
│  (게시물 본문)              │  ┌──────────────┐  │
│                           │  │ 댓글  │ Q&A  │  │
│                           │  └──────────────┘  │
│                           │                     │
└───────────────────────────┴─────────────────────┘
```

- **메인 콘텐츠 (왼쪽)** — 게시물 본문을 표시합니다.
- **댓글 사이드바 (오른쪽)** — 두 개의 탭으로 구성됩니다.
  - **댓글 탭** — 게시물에 달린 댓글을 확인하고 작성합니다.
  - **Q&A 탭** — 게시물 내용을 바탕으로 AI가 생성한 질문과 답변을 확인합니다.

---

## 반응형 디자인 (모바일)

모바일 기기에서는 사이드바가 자동으로 접힙니다.

- 상단 헤더의 **사이드바 토글** 버튼을 탭하면 사이드바가 열립니다.
- 사이드바 외부 영역을 탭하면 사이드바가 다시 닫힙니다.
- 게시물 읽기 화면에서는 댓글 사이드바가 본문 아래에 배치됩니다.

> **Tip:** 모바일에서 더 편한 독서 경험을 원한다면 사이드바를 닫고 메인 콘텐츠에 집중하세요.

# 회원가입 및 로그인

Skillnote는 두 가지 방법으로 가입하고 로그인할 수 있습니다.

---

## Google로 시작하기 (권장)

가장 빠르고 간편한 방법입니다.

1. Skillnote 홈페이지에 접속합니다
2. 로그인 모달에서 **"Google로 시작하기"** 버튼을 클릭합니다
3. Google 계정을 선택하거나 로그인합니다
4. 자동으로 Skillnote에 가입 및 로그인됩니다

> **Tip:** Google 로그인을 사용하면 별도의 비밀번호 관리가 필요 없어 편리합니다.

---

## 이메일로 가입하기

Google 계정 없이 이메일과 비밀번호로 가입할 수도 있습니다.

### 가입 절차

1. 로그인 모달에서 **"이메일로 가입"** 탭을 선택합니다
2. 다음 정보를 입력합니다:
   - **이메일 주소**: 유효한 이메일 주소
   - **비밀번호**: 안전한 비밀번호
   - **비밀번호 확인**: 동일한 비밀번호 재입력
3. **"가입하기"** 버튼을 클릭합니다
4. 입력한 이메일로 **인증 메일**이 발송됩니다
5. 이메일에서 인증 링크를 클릭하여 가입을 완료합니다

> **주의:** 이메일 인증을 완료해야 모든 기능을 사용할 수 있습니다. 인증 메일이 도착하지 않으면 스팸 폴더를 확인해 주세요.

### 이메일로 로그인

1. 로그인 모달에서 이메일과 비밀번호를 입력합니다
2. **"로그인"** 버튼을 클릭합니다

---

## 비밀번호 재설정

비밀번호를 잊어버린 경우 다음 절차로 재설정할 수 있습니다:

1. 로그인 모달에서 **"비밀번호를 잊으셨나요?"** 를 클릭합니다
2. 가입 시 사용한 **이메일 주소**를 입력합니다
3. 이메일로 발송된 **인증 코드**를 입력합니다
4. **새 비밀번호**를 설정합니다
5. 비밀번호 변경 완료 후 새 비밀번호로 로그인합니다

---

## 자동 로그인 안내

- 로그인 상태는 브라우저에 저장됩니다
- 인증 토큰은 자동으로 갱신되어 별도의 재로그인 없이 사용할 수 있습니다
- 세션이 만료되면 자동으로 재로그인 안내 대화상자가 표시됩니다
- 보호된 페이지에 접근하면 로그인 모달이 자동으로 나타납니다

---

## 계정 보안 팁

- 강력한 비밀번호를 사용하세요 (영문, 숫자, 특수문자 조합)
- Google 로그인을 사용하면 Google의 보안 기능을 함께 활용할 수 있습니다
- 공용 컴퓨터에서 사용 후 반드시 로그아웃하세요

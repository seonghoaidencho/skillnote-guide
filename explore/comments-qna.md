# 댓글 및 Q&A

게시물을 읽을 때 오른쪽 댓글 사이드바에서 댓글을 남기거나 AI 생성 질문에 대해 토론할 수 있습니다.

---

## 댓글 사이드바

게시물 상세 페이지의 오른쪽에 **댓글 사이드바**가 표시됩니다. 두 개의 탭으로 구성되어 있습니다:

- **댓글 탭**: 게시물에 대한 일반 댓글
- **Q&A 탭**: AI가 생성한 질문과 관련 토론

---

## 댓글

### 댓글 작성

1. 댓글 사이드바에서 **"댓글"** 탭을 선택합니다
2. 하단의 입력창에 댓글을 작성합니다
3. **"등록"** 버튼을 클릭하여 댓글을 게시합니다

### 댓글 기능

- **좋아요/싫어요**: 댓글에 반응을 표시할 수 있습니다
- **스레드**: 댓글에 대한 답글을 통해 대화를 이어갈 수 있습니다

---

## Q&A

Q&A 탭에서는 게시물 내용을 바탕으로 **AI가 자동 생성한 질문**을 확인하고 토론할 수 있습니다.

### Q&A 카드 구성

각 질문은 카드 형태로 표시되며, 다음 정보를 포함합니다:

| 항목 | 설명 |
|------|------|
| **복습 힌트** | 간격 반복 학습 일정 (예: "1일차 복습", "1주차 복습") |
| **질문 유형** | 질문의 형태 (참/거짓, 객관식, 빈칸 채우기 등) |
| **난이도** | 1~5 레벨로 표시되는 난이도 |
| **태그** | 관련 주제 태그 |
| **질문 내용** | 질문 텍스트 |
| **기반 문구** | 질문이 생성된 원본 텍스트 요약 |

### 복습 힌트 일정

질문의 복습 힌트는 게시물 작성일로부터의 경과 기간에 따라 결정됩니다:

| 경과 기간 | 힌트 |
|-----------|------|
| 1일 이내 | 1일차 복습 |
| 3일 이내 | 3일차 복습 |
| 7일 이내 | 1주차 복습 |
| 14일 이내 | 2주차 복습 |
| 30일 이내 | 1개월차 복습 |
| 그 이상 | N개월차 복습 |

### Q&A 상세 보기

질문 카드를 클릭하면 상세 페이지로 이동합니다:

- **질문 전체 내용**이 상단에 표시됩니다
- 하단에 **댓글 패널**이 표시되어 해당 질문에 대해 토론할 수 있습니다
- 댓글 수와 북마크 수가 표시됩니다
- 뒤로가기 버튼으로 Q&A 목록으로 돌아갈 수 있습니다

### Q&A에 참여하기

1. Q&A 탭에서 관심 있는 질문 카드를 클릭합니다
2. 상세 페이지에서 댓글을 작성하여 답변이나 의견을 남깁니다
3. 다른 사용자의 답변도 확인하고 토론에 참여할 수 있습니다

> **Tip:** Q&A에 직접 답변을 작성하는 것도 훌륭한 복습 방법입니다. 남에게 설명하는 과정에서 이해도가 깊어집니다.

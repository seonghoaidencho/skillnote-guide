# 댓글 및 Q&A

게시물을 읽을 때 화면 오른쪽에 **댓글 사이드바**가 표시됩니다. 이 패널에서 게시물에 반응을 남기거나, 댓글로 소통하거나, AI가 생성한 복습 질문을 확인할 수 있습니다.

---

## 댓글 사이드바 기본 구조

댓글 사이드바는 화면 오른쪽 고정 영역에 표시됩니다.

- **너비**: 화면 너비의 34vw (최소 280px, 최대 560px)
- **모바일**: 전체 화면 오버레이로 표시됩니다.
- **토글 버튼**: 패널 아이콘(열기) 또는 X 아이콘(닫기)이 항상 화면에 표시되어 있어 언제든지 패널을 열고 닫을 수 있습니다.

### 세 가지 탭

| 탭 이름 | 설명 |
|---------|------|
| **정보** | 게시물 좋아요, 공유, 작성자 정보 |
| **댓글** | 댓글 목록 및 작성 (댓글 수 배지 — 회색) |
| **복습** | AI 생성 복습 질문 목록 (질문 수 배지 — 빨간색, 학습 기록 게시물에만 표시) |

## 정보 탭

### 좋아요 및 공유

- **좋아요 버튼** (하트 아이콘): 클릭하면 게시물에 좋아요를 표시합니다. 현재 좋아요 수가 버튼 옆에 표시됩니다.
- **공유 버튼**: 게시물 링크를 공유합니다.

### 더보기 메뉴 ("...")

게시물 오른쪽 상단의 **"..."** 버튼을 클릭하면 추가 옵션 메뉴가 표시됩니다.

- **팔로우 / 언팔로우**: 작성자를 팔로우하거나 팔로우를 취소합니다.
- **신고**: 부적절한 게시물을 신고합니다.

### 작성자 정보

정보 탭 하단에는 게시물 작성자의 이름과 아바타가 표시됩니다. 작성자 이름이나 아바타에 마우스를 올리면 **UserProfileCard 팝오버**가 나타나 작성자의 프로필 요약을 바로 확인할 수 있습니다.

## 댓글 탭

### 댓글 정렬

댓글 목록 상단의 **"정렬:"** 드롭다운에서 댓글 표시 순서를 변경할 수 있습니다.

| 옵션 | 설명 |
|------|------|
| **최신 순** | 가장 최근에 작성된 댓글이 위에 표시됩니다 |
| **오래된 순** | 가장 먼저 작성된 댓글이 위에 표시됩니다 |
| **인기 높은 순** | 좋아요를 많이 받은 댓글이 위에 표시됩니다 |

### 댓글 목록

- 댓글은 중첩 답글(스레드) 형태로 표시됩니다.
- 삭제된 댓글은 **"해당 댓글은 삭제되었습니다."** 라는 텍스트로 자리를 대신합니다.
- 수정된 댓글에는 **(수정됨)** 라벨이 표시됩니다.

### 댓글 액션

각 댓글에서 다음 작업을 수행할 수 있습니다.

- **좋아요**: 댓글에 공감을 표시합니다.
- **답글**: 해당 댓글에 대한 답글을 작성합니다.
- **수정**: 내가 작성한 댓글을 수정합니다.
- **삭제**: 내가 작성한 댓글을 삭제합니다.

### 댓글 작성 에디터

댓글 탭 하단에 리치 텍스트 에디터가 표시됩니다.

- **입력창 플레이스홀더**: "댓글을 입력하세요…"
- **미니 툴바** (에디터 상단에 표시되는 서식 버튼):

| 버튼 | 기능 |
|------|------|
| **B** (Bold) | 굵게 |
| **I** (Italic) | 기울임 |
| **U** (Underline) | 밑줄 |
| **S** (Strike) | 취소선 |
| **<>** (Code) | 인라인 코드 |
| 목록 (Bullet) | 글머리 기호 목록 |
| 번호 (Ordered) | 번호 매기기 목록 |
| 인용 (Quote) | 인용 블록 |
| 실행 취소 (Undo) | 이전 상태로 되돌리기 |
| 다시 실행 (Redo) | 취소한 작업 다시 실행 |

### 댓글이 없을 때

아직 작성된 댓글이 없는 경우 다음 안내가 표시됩니다.

> 아직 댓글이 없습니다.
>
> 첫 댓글을 남겨보세요!

## 복습 탭 (Q&A)

복습 탭은 **학습 기록 유형**의 게시물에만 표시됩니다. AI가 게시물 내용을 분석해 자동으로 생성한 복습 질문들을 여기서 확인할 수 있습니다.

### 로딩 및 빈 상태

- 질문을 불러오는 중에는 스피너와 함께 **"질문을 불러오는 중..."** 메시지가 표시됩니다.
- 생성된 질문이 아직 없는 경우 **"아직 질문이 없습니다."** 메시지가 표시됩니다.

### Q&A 카드 구성

각 질문은 카드 형태로 표시되며, 다음 정보를 포함합니다:

| 항목 | 설명 |
|------|------|
| **복습 힌트** | 게시물 작성일 기준 경과 기간에 따른 복습 단계 표시 |
| **유형 배지** | 질문의 형태 (예: 객관식, 빈칸 채우기 등) |
| **난이도 배지** | 1~5 레벨로 표시되는 난이도 |
| **태그** | 질문이 속한 주제 태그 |
| **질문 텍스트** | 실제 질문 내용 |
| **basePhrases** | 질문이 생성된 원본 텍스트(기반 문구) |

### 복습 힌트 일정

복습 힌트는 게시물 작성일로부터 경과한 기간에 따라 자동으로 결정됩니다.

| 경과 기간 | 복습 힌트 |
|-----------|----------|
| 1일 이내 | 1일차 복습 |
| 3일 이내 | 3일차 복습 |
| 7일 이내 | 1주차 복습 |
| 14일 이내 | 2주차 복습 |
| 30일 이내 | 1개월차 복습 |
| 30일 초과 | N개월차 복습 |

### Q&A 상세 보기

질문 카드를 클릭하면 해당 질문의 상세 화면으로 이동합니다.

- 상단의 **"질문 목록으로 돌아가기"** 버튼을 클릭하면 복습 탭의 카드 목록으로 돌아갑니다.
- 질문 전체 내용이 표시됩니다.
- 화면 하단의 **댓글 패널**에서 해당 질문에 대한 의견을 나누거나 다른 사람의 답변을 확인할 수 있습니다.

> **Tip:** 복습 질문에 직접 답변을 댓글로 작성해 보세요. 남에게 설명하듯 글로 정리하는 과정이 가장 효과적인 복습 방법 중 하나입니다.

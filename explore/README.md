# 탐색하기

Skillnote는 개인 학습 도구일 뿐만 아니라 다른 개발자들과 지식을 나누는 **학습 커뮤니티**이기도 합니다. 탐색 기능을 통해 다른 사용자들의 학습 게시물을 발견하고, 블로그를 방문하고, 댓글과 Q&A로 소통할 수 있습니다.

| [사용자 블로그](user-blogs.md) | 다른 사용자의 블로그 방문 및 팔로우 |

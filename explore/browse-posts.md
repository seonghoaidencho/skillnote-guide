# 게시물 둘러보기

탐색 페이지에서는 Skillnote 커뮤니티의 다양한 학습 콘텐츠를 발견하고 영감을 얻을 수 있습니다. 사이드바에서 **"탐색"** 메뉴를 클릭하면 이동할 수 있으며, 페이지 최상단에는 다음과 같은 헤더가 표시됩니다.

> **탐색** — 다양한 학습 콘텐츠를 발견하고 영감을 얻어보세요

---

## 탐색 페이지 구성

탐색 페이지는 세 개의 섹션으로 나뉩니다.

---

### 섹션 1 — 트렌딩

트렌딩(trendingUp 아이콘) 섹션에는 현재 주목받고 있는 게시물이 큰 카드(PostCardLarge) 형식으로 표시됩니다. 카드에는 게시물 제목, 설명, 작성자 아바타와 이름, 작성일, 태그가 한눈에 보입니다.

- 섹션 오른쪽 상단의 **"전체보기"** 버튼을 클릭하면 트렌딩 게시물 전체 목록 화면으로 이동합니다.

---

### 섹션 2 — 최신 학습 노트

최신 학습 노트(clock 아이콘) 섹션에는 최근 작성된 게시물이 **태그별 컬럼** 구조로 표시됩니다.

- 최대 3개의 태그 컬럼이 나란히 배치됩니다.
- 각 컬럼 상단에는 `# TagName` 형식의 태그 배지가 표시되고, 오른쪽에 **"전체보기"** 버튼이 있습니다.
- 각 컬럼 안에는 소형 카드(PostCardSmall) 형식으로 최대 3개의 게시물이 표시됩니다.

---

### 섹션 3 — 인기 태그

인기 태그(tag 아이콘) 섹션에는 Skillnote 전체에서 자주 사용되는 태그들이 목록으로 표시됩니다.

- 각 태그 버튼은 `# TagName` 형식으로 표시되며, 태그 옆에 해당 태그를 사용한 게시물 수(count)가 함께 나타납니다.
- 태그를 클릭하면 해당 태그로 필터링된 전체보기 화면으로 이동합니다.

---

## 전체보기 화면

**"전체보기"** 버튼이나 인기 태그를 클릭하면 해당 태그의 게시물만 모아볼 수 있는 전체보기 화면이 열립니다.

### 상단 영역

| 요소 | 설명 |
|------|------|
| **탐색으로 돌아가기** | 뒤로가기 버튼. 클릭하면 탐색 메인 페이지로 돌아갑니다. |
| 태그 아이콘 + 태그 제목 | 현재 보고 있는 태그 이름이 대문자로 표시됩니다. |
| **{N}개의 학습 기록** | 해당 태그가 달린 게시물의 총 수입니다. |
| **최신순** | 정렬 버튼. 현재는 최신순으로 정렬됩니다. |

### 연관 태그

현재 태그와 함께 자주 사용되는 연관 태그들이 배지 목록으로 표시됩니다. 각 배지를 클릭하면 해당 태그의 전체보기 화면으로 바로 이동할 수 있습니다.

### 게시물 그리드

게시물은 카드 형태로 그리드 배치됩니다. 각 카드에는 다음 정보가 표시됩니다:

- 게시물 제목 및 요약 설명
- 작성자 아바타와 이름
- 작성일
- 태그 목록 (현재 보고 있는 태그는 강조 표시됨)

### 게시물이 없을 때

해당 태그로 작성된 게시물이 아직 없는 경우 다음 안내 메시지와 함께 버튼이 표시됩니다.

> 아직 게시물이 없어요
>
> 이 태그로 첫 번째 학습 기록을 작성해보세요!

- **"새 글 작성하기"** 버튼을 클릭하면 에디터로 이동하여 해당 태그로 게시물을 직접 작성할 수 있습니다.

---

## 게시물 읽기

게시물 카드를 클릭하면 상세 페이지로 이동합니다.

- **본문**: 화면 왼쪽(또는 전체 영역)에 게시물 내용이 렌더링됩니다.
- **댓글 사이드바**: 화면 오른쪽에 정보, 댓글, 복습 탭이 있는 패널이 열립니다. 자세한 내용은 [댓글 및 Q&A](./comments-qna.md) 문서를 참고하세요.

> **Tip:** 흥미로운 게시물을 발견하면 작성자 이름을 클릭해 블로그를 방문해 보세요. 같은 주제의 더 많은 학습 기록을 한 번에 확인할 수 있습니다.

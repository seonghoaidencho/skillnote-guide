# 게시물 둘러보기

탐색 페이지에서는 다른 사용자들이 공개한 학습 게시물을 발견할 수 있습니다.

---

## 탐색 페이지 접근

사이드바에서 **"탐색"** 메뉴를 클릭하면 탐색 페이지(`/explore`)로 이동합니다.

---

## 트렌딩 게시물

탐색 페이지 상단에는 **인기 게시물**이 표시됩니다. 조회수, 댓글 수 등을 기반으로 주목받는 게시물을 확인할 수 있습니다.

각 게시물 카드에는 다음 정보가 표시됩니다:

| 항목 | 설명 |
|------|------|
| **제목** | 게시물 제목 |
| **작성자** | 작성자 이름과 프로필 이미지 |
| **태그** | 게시물에 지정된 주제 태그 |
| **작성일** | 게시물이 발행된 날짜 |

---

## 태그별 탐색

인기 태그 목록에서 관심 있는 태그를 클릭하면 해당 태그가 포함된 게시물만 필터링하여 볼 수 있습니다.

### 인기 태그

탐색 페이지에 자주 사용되는 태그들이 표시됩니다. 예를 들어:

- `JavaScript`, `Python`, `React`, `TypeScript`
- `알고리즘`, `자료구조`, `데이터베이스`
- `DevOps`, `클라우드`, `AI/ML`

### 태그 필터

1. 관심 있는 태그를 클릭합니다
2. 해당 태그가 포함된 게시물 목록이 표시됩니다
3. 다른 태그를 클릭하여 필터를 변경할 수 있습니다

---

## 게시물 읽기

게시물 카드를 클릭하면 상세 페이지로 이동합니다. 게시물 읽기 화면에서는:

- **본문**: 왼쪽에 게시물 전체 내용이 표시됩니다
- **댓글 사이드바**: 오른쪽에 댓글과 Q&A 탭이 표시됩니다

> **Tip:** 흥미로운 게시물을 발견하면 작성자의 블로그를 방문하여 더 많은 글을 확인해 보세요.

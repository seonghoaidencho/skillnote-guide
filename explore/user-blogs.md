# 사용자 블로그

Skillnote의 모든 사용자는 자신만의 블로그 페이지를 가집니다. 공개된 게시물은 작성자의 블로그 페이지에서 한 곳에 모아볼 수 있습니다.

---

## 블로그 방문하기

다른 사용자의 블로그에 접근하는 방법은 두 가지입니다.

- **게시물에서**: 게시물 본문 또는 댓글 사이드바 **"정보"** 탭에 표시된 작성자 이름이나 아바타를 클릭합니다.
- **탐색 페이지에서**: 게시물 카드의 작성자 이름을 클릭합니다.
- **URL 직접 입력**: 브라우저 주소창에 `skillnote.ai/{유저태그}` 형식으로 입력합니다.

---

## 블로그 페이지 구성

### 프로필 헤더

블로그 페이지 상단에는 해당 사용자의 프로필 정보가 표시됩니다.

| 항목 | 설명 |
|------|------|
| **프로필 이미지** | 사용자가 설정한 아바타. 미설정 시 블로그 이름 첫 글자 표시 |
| **플랜 배지** | Plus 또는 Pro 플랜 사용자에게 표시되는 구독 배지 |
| **블로그 이름** | 프로필 설정에서 지정한 블로그 이름 |
| **유저태그** | 고유 식별자 (변경 불가) |
| **소개** | 프로필 설정에서 작성한 자기소개 |
| **소개 링크** | GitHub, LinkedIn, 개인 웹사이트 등 등록된 링크 |
| **팔로워 / 팔로잉 수** | 숫자를 클릭하면 목록을 확인할 수 있습니다 |

### 게시물 목록

프로필 헤더 아래에 해당 사용자가 공개한 모든 게시물이 카드 형태로 나열됩니다.

- 기본 정렬은 **최신순**입니다.
- 각 카드에는 게시물 제목, 요약 설명, 태그, 작성일이 표시됩니다.
- 카드를 클릭하면 해당 게시물 상세 페이지로 이동합니다.

---

## 팔로우

관심 있는 사용자를 팔로우하면 해당 사용자의 활동을 쉽게 추적할 수 있습니다.

### 팔로우하기

1. 팔로우하고 싶은 사용자의 블로그 페이지를 방문합니다.
2. 프로필 헤더의 **"팔로우"** 버튼을 클릭합니다.

또는 게시물 상세 페이지의 댓글 사이드바 **"정보"** 탭 → **"..."** 메뉴에서 **"팔로우"** 를 선택할 수도 있습니다.

### 팔로우 취소

1. 이미 팔로우 중인 사용자의 블로그 페이지를 방문합니다.
2. **"언팔로우"** 버튼을 클릭합니다.

또는 댓글 사이드바 **"정보"** 탭 → **"..."** 메뉴에서 **"언팔로우"** 를 선택해도 됩니다.

### 팔로워 / 팔로잉 목록

블로그 프로필 헤더에 표시된 팔로워 수 또는 팔로잉 수를 클릭하면 해당 목록을 확인할 수 있습니다. 목록에서 사용자 이름을 클릭하면 해당 사용자의 블로그로 바로 이동합니다.

---

## 작성자 프로필 팝오버

게시물을 읽는 도중 작성자 이름이나 아바타에 마우스를 올리면 **UserProfileCard 팝오버**가 표시됩니다. 페이지를 이동하지 않고도 다음 정보를 빠르게 확인할 수 있습니다.

- 프로필 이미지 및 블로그 이름
- 소개글
- 팔로워 / 팔로잉 수
- 팔로우 / 언팔로우 버튼

> **Tip:** 비슷한 분야를 공부하는 사용자를 팔로우하면 새로운 학습 관점을 얻고 꾸준히 학습하는 데 동기부여가 됩니다. 탐색 페이지의 트렌딩 게시물에서 마음에 드는 작성자를 발견해 보세요.

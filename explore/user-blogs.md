# 사용자 블로그

Skillnote에서 공개된 게시물은 각 사용자의 개인 블로그 페이지에서 모아볼 수 있습니다.

---

## 블로그 방문하기

사용자 블로그에 접근하는 방법:

- **게시물에서**: 게시물 작성자의 이름이나 프로필 이미지를 클릭합니다
- **URL 직접 입력**: `skillnote.ai/{사용자ID}` 형식으로 접근합니다

---

## 블로그 구성

### 프로필 헤더

블로그 상단에는 사용자의 프로필 정보가 표시됩니다:

- **프로필 이미지**: 사용자가 설정한 이미지 또는 기본 아바타
- **표시 이름**: 사용자 이름
- **소셜 링크**: 사용자가 등록한 GitHub, Twitter 등의 링크

### 게시물 목록

프로필 아래에 해당 사용자가 공개한 모든 게시물이 목록으로 표시됩니다.

- 최신순으로 정렬됩니다
- 각 게시물의 제목, 태그, 작성일이 표시됩니다

---

## 팔로우

관심 있는 사용자를 팔로우하여 활동을 추적할 수 있습니다.

### 팔로우하기

1. 사용자의 블로그 페이지를 방문합니다
2. **"팔로우"** 버튼을 클릭합니다

### 팔로우 취소

1. 이미 팔로우 중인 사용자의 블로그를 방문합니다
2. **"팔로우 취소"** 버튼을 클릭합니다

### 팔로워 목록

프로필에서 팔로워 수와 팔로잉 수를 확인할 수 있으며, 클릭하면 목록을 볼 수 있습니다.

> **Tip:** 같은 분야를 공부하는 사용자를 팔로우하면 학습에 대한 동기부여를 얻을 수 있습니다.

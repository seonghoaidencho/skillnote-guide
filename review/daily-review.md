# 오늘의 복습

매일 Skillnote 홈 대시보드에는 오늘 복습할 내용이 요약된 카드가 표시됩니다. 몇 분만 투자하면 학습한 내용을 오래 기억할 수 있습니다.

---

## 1단계: 대시보드에서 복습 카드 확인

홈 화면의 **오늘의 복습** 카드에는 다음 정보가 표시됩니다.

- **문제 수**: 오늘 복습해야 할 총 문제 개수
- **출처 포스트 수**: 문제가 생성된 TIL 포스트 개수

**복습 시작** 버튼을 클릭하면 `/questions` 페이지로 이동합니다.

> **Tip:** 복습 카드에 문제가 없는 날은 오늘 복습이 필요 없는 날입니다. 꾸준히 TIL을 작성하면 복습 주기가 자동으로 생성됩니다.

---

## 2단계: 커버 페이지에서 세션 개요 확인

복습 세션 시작 전, 커버 페이지에서 오늘의 복습 내용을 한눈에 확인할 수 있습니다.

- **출처 포스트별 문제 그룹**: 어떤 포스트에서 몇 개의 문제가 나왔는지 확인
- **D+N 표시**: 포스트 작성 후 며칠이 지났는지 표시 (예: D+7은 작성 후 7일째)
- **진행 현황**: 이미 답변한 문제와 남은 문제 수 표시

모든 내용을 확인했으면 **Start Review** 버튼을 클릭해 세션을 시작합니다.

---

## 3단계: 문제 풀기

복습 세션에서는 한 번에 한 문제씩 풀어나갑니다.

### 화면 구성

- **상단 진행 바**: 현재 몇 번째 문제인지, 전체 진행률을 색상으로 표시
- **문제 카드**: 문제 유형, 난이도, 출처 포스트 제목, 문제 내용
- **답변 영역**: 문제 유형에 맞는 입력 방식으로 답변 작성

### 문제 간 이동

- **이전 버튼 (←)**: 이전 문제로 돌아가 답변을 수정할 수 있습니다.
- **다음 버튼 (→)**: 다음 문제로 넘어갑니다.

> **Tip:** 답변을 입력하지 않고 넘어갈 수 있지만, 빈 답변은 오답으로 처리됩니다. 확실하지 않아도 최선을 다해 작성해 보세요.

---

## 4단계: 복습 완료

마지막 문제에서 답변을 작성한 후 **복습 완료** 버튼을 클릭합니다.

### 채점 단계

모든 답변이 AI에게 일괄 제출되어 채점이 진행됩니다. 문제 수에 따라 수 초가 소요될 수 있습니다.

채점이 완료되면 자동으로 **결과 페이지**로 이동합니다.

> **Tip:** 채점 중 페이지를 닫아도 괜찮습니다. 채점 결과는 저장되며, 나중에 결과 페이지에서 다시 확인할 수 있습니다.

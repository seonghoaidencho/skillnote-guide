# 복습 시스템

Skillnote의 복습 시스템은 **간격 반복 학습(Spaced Repetition)** 원리를 기반으로, 여러분이 작성한 TIL 포스트에서 AI가 자동으로 문제를 생성합니다. 매일 꾸준한 복습을 통해 학습한 내용을 장기 기억으로 전환할 수 있습니다.

---

## 핵심 개념: 간격 반복 학습

간격 반복 학습은 **기억이 희미해지기 직전에 반복 학습**하는 방식입니다. 처음에는 짧은 간격으로, 시간이 지날수록 점점 긴 간격으로 복습하면서 장기 기억을 효율적으로 형성합니다.

```
학습 직후 → 1일 후 → 3일 후 → 7일 후 → 14일 후 → 30일 후 ...
```

단순히 같은 내용을 여러 번 읽는 것보다 훨씬 효과적으로 지식을 내재화할 수 있습니다.

---

## Skillnote 복습 시스템의 동작 방식

1. **자동 문제 생성**: TIL 포스트를 작성하면 AI가 핵심 내용을 분석하여 다양한 유형의 문제를 자동으로 생성합니다.
2. **일일 복습 세션**: 홈 대시보드에서 오늘 복습해야 할 문제 수를 확인하고, 짧은 세션으로 복습을 완료합니다.
3. **AI 채점**: 작성한 답변을 AI가 의미적으로 평가하여 정확한 피드백을 제공합니다.
4. **결과 확인**: 복습이 끝나면 원본 포스트와 함께 결과를 확인하고 부족한 부분을 파악합니다.

> **Tip:** 매일 조금씩 복습하는 것이 핵심입니다. Skillnote는 학습 부담을 최소화하면서 최대 효과를 낼 수 있도록 복습량을 자동으로 조절합니다.

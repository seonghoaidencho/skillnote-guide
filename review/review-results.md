# 복습 결과 확인

복습 세션이 완료되고 채점이 끝나면 자동으로 결과 페이지로 이동합니다. 결과 페이지에서는 이번 세션의 전체 성과를 한눈에 파악하고, 각 질문의 정오 여부와 AI 피드백을 확인하며, 원본 게시물에서 해당 내용이 어디서 왔는지 바로 조회할 수 있습니다.

---

## 결과 헤더

페이지 최상단에 이번 세션의 요약 정보가 가로 한 줄로 표시됩니다.

- 초록색 체크 원형(CheckCircle) 아이콘과 함께 **"복습 완료!"** 텍스트가 표시됩니다.
- 그 옆에 인라인 통계가 이어집니다.
  - **{elapsed}**: 세션에 소요된 시간
  - **{N}문제**: 이번 세션에서 푼 총 문제 수
  - **{accuracy}% 정답**: 정답률
- 헤더 우측에 **"처음으로"** 버튼이 있어 커버 페이지로 돌아갈 수 있습니다.

---

## 2단 분할 레이아웃 (데스크탑)

데스크탑 환경에서는 화면이 좌우 두 개의 패널로 나뉩니다. 두 패널은 독립적으로 스크롤됩니다.

```
┌──────────────────────────┬──────────────────────────┐
│                          │                          │
│   왼쪽 패널              │   오른쪽 패널            │
│   "학습 자료"            │   "질문별 결과"          │
│   (bookOpen 아이콘)      │   "{N}문제" 카운트       │
│                          │                          │
│   선택된 질문의          │   각 질문 결과 항목      │
│   원본 게시물 내용       │   (클릭 시 왼쪽 연동)    │
│   (핵심 문구 하이라이트) │                          │
│                          │                          │
└──────────────────────────┴──────────────────────────┘
```

---

## 왼쪽 패널 — 학습 자료

패널 제목은 bookOpen 아이콘과 함께 **"학습 자료"**로 표시됩니다.

### 빈 상태 (질문 미선택)

오른쪽 패널에서 아무 질문도 선택하지 않은 초기 상태입니다.

- 책 아이콘과 함께 **"질문을 선택해 학습자료를 살펴보세요"** 안내 문구가 표시됩니다.

### 질문 선택 후

오른쪽 패널에서 질문 항목을 클릭하면 왼쪽 패널에 해당 질문의 원본 게시물 내용이 표시됩니다.

- **게시물 제목**이 패널 상단에 표시됩니다.
- 현재 선택한 질문 번호를 나타내는 **"Q{N}"** 배지가 제목 옆에 표시됩니다.
- 그 아래에 원본 게시물의 전체 내용이 읽기 전용 에디터 뷰로 렌더링됩니다.
- AI가 이 질문을 생성할 때 참조한 원본 문구(basePhrases)가 본문 내에서 **하이라이트** 처리되어 표시됩니다.

하이라이트된 문구를 통해 해당 질문이 게시물의 어느 부분에서 나왔는지 정확히 파악할 수 있습니다. 오답인 질문을 클릭하여 원본 맥락을 다시 읽으면 개념을 정확히 이해하는 데 효과적입니다.

---

## 오른쪽 패널 — 질문별 결과

패널 제목은 **"질문별 결과"**이며, 제목 옆에 이번 세션의 총 문제 수인 **"{N}문제"** 카운트가 표시됩니다.

### 각 결과 항목 구성

결과 항목을 클릭하면 왼쪽 패널에 해당 질문의 학습 자료가 로드됩니다.

**기본 정보**

- **번호 배지**: 세션 내 질문 순서 번호입니다.
- **질문 유형 레이블**: OX 퀴즈, 객관식, 빈칸 채우기, 단답형, 적용형 중 해당 유형이 표시됩니다.
- **정오 아이콘**: 채점 가능한 유형의 경우 정답이면 **정답 표시 (초록색)**, 오답이면 **오답 표시 (빨간색)**으로 표시됩니다.
- **질문 텍스트**: 실제 질문 내용이 표시됩니다.

**적용형 전용**

- **점수 배지**: **"{score}점"** 형태로 0~100점 사이의 AI 채점 점수가 표시됩니다.

**창의적 답변**

- `is_gradable`이 `false`인 답변의 경우 **"창의적 답변"** 배지가 표시됩니다.
- AI가 정오를 명확히 판단하기 어려운 열린 답변에 부여됩니다. 오답으로 처리되지 않으며, 내용을 직접 확인하고 스스로 평가할 수 있습니다.

**답변 및 정답 표시**

- **"나의 답변: {answer}"**: 내가 제출한 답변이 표시됩니다. 오답인 경우 위험(danger) 색상으로 강조됩니다.
- **"정답: {answer}"**: 올바른 정답이 초록색으로 표시됩니다.

**동의어 인정 배지**

- 내가 제출한 답변이 정확한 표현과 다르더라도 동의어로 인정된 경우 **"동의어로 정답 처리"** 배지가 표시됩니다.

**AI 피드백**

- **"SkillNote 피드백: {feedback}"**: AI가 해당 답변에 대해 작성한 상세 피드백이 표시됩니다. 단답형과 적용형 질문에서 제공됩니다.

## 모바일 레이아웃

모바일 환경에서는 2단 분할 대신 단일 세로 스크롤 레이아웃으로 표시됩니다.

- 질문별 결과 목록이 화면 전체에 표시됩니다.
- 결과 항목을 클릭하면 **바텀 시트(bottom sheet)** 형태로 학습 자료가 올라옵니다.
  - 시트 상단에 드래그 핸들 바가 표시됩니다.
  - bookOpen 아이콘과 **"학습 자료"** 제목이 표시됩니다.
  - 우측 상단의 **X** 버튼을 눌러 바텀 시트를 닫을 수 있습니다.
  - 시트 내부에서 원본 게시물 내용과 하이라이트된 핵심 문구를 확인할 수 있습니다.

## 결과 항목 요소 요약

| 요소 | 표시 조건 | 설명 |
|------|-----------|------|
| 번호 배지 | 항상 | 세션 내 질문 순서 |
| 질문 유형 레이블 | 항상 | OX 퀴즈, 객관식 등 |
| 정오 아이콘 | 채점 가능한 유형 | 초록(정답) / 빨간(오답) |
| "{score}점" 배지 | 적용형 | AI 채점 점수 (0~100) |
| "창의적 답변" 배지 | `is_gradable === false` | AI가 정오 판단 불가인 답변 |
| "나의 답변: {answer}" | 항상 | 오답 시 위험 색상 강조 |
| "정답: {answer}" | 항상 | 초록색으로 표시 |
| "동의어로 정답 처리" 배지 | 동의어 인정 시 | 유사 표현이 정답 처리된 경우 |
| "SkillNote 피드백: {feedback}" | 단답형 / 적용형 | AI가 작성한 상세 피드백 |

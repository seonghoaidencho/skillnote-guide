# 복습 결과 확인

복습 세션이 완료되고 채점이 끝나면 결과 페이지로 이동합니다. 결과 페이지에서는 각 문제의 정오 여부와 AI 피드백을 확인하고, 원본 포스트에서 해당 내용이 어디에서 왔는지 바로 확인할 수 있습니다.

---

## 결과 페이지 레이아웃

결과 페이지는 **2단 분할 화면**으로 구성됩니다.

```
┌──────────────────────┬──────────────────────┐
│                      │                      │
│   왼쪽 패널 (50%)    │   오른쪽 패널 (50%)  │
│                      │                      │
│  원본 포스트 내용    │   문제 결과 목록     │
│  (기반 문구 하이라이트)│                     │
│                      │                      │
└──────────────────────┴──────────────────────┘
```

두 패널은 **독립적으로 스크롤**됩니다. 오른쪽에서 문제 결과를 확인하면서 왼쪽의 포스트를 별도로 스크롤할 수 있습니다.

---

## 결과 헤더

결과 목록 상단에 복습 세션 전체 요약이 표시됩니다.

| 항목 | 설명 |
|------|------|
| **완료 시각** | 복습 세션을 완료한 시간 |
| **총 문제 수** | 이번 세션의 전체 문제 개수 |
| **정답률** | 정답 수 / 전체 문제 수 (%) |

---

## 문제 결과 항목

각 문제 결과 항목에는 다음 정보가 표시됩니다.

### 기본 정보

- **문제 번호**: 세션에서의 순서
- **문제 유형 배지**: O/X, 객관식, 빈칸 채우기, 단답형, 서술형
- **정오 표시**: 정답(초록색 ✅) 또는 오답(빨간색 ❌)

### 자동 채점 문제 (O/X, 객관식, 빈칸 채우기)

- **정답**: 올바른 정답 표시
- **내 답변**: 오답인 경우 내가 제출한 답변 표시 (정답인 경우 생략)

### AI 채점 문제 (단답형, 서술형)

- **점수 배지**: 서술형의 경우 0~100점 점수 표시
- **AI 피드백**: AI가 작성한 상세한 평가 내용
- **내 답변**: 제출한 답변 전문

### 특수 배지

| 배지 | 의미 |
|------|------|
| **동의어 인정** | 동의어나 유사 표현을 정답으로 인정한 경우 |
| **창의적 답변** | AI가 정오를 명확히 판단하기 어려운 경우 |

---

## 원본 포스트 연동

오른쪽 패널에서 특정 문제 결과를 클릭하면, 왼쪽 패널의 원본 포스트에서 해당 문제의 **기반 문구(basePhrases)**가 강조 표시됩니다.

- 기반 문구는 AI가 문제를 생성할 때 참조한 원본 포스트의 구체적인 문장이나 구절입니다.
- 강조 표시를 통해 해당 문제가 포스트의 어느 부분에서 나왔는지 즉시 확인할 수 있습니다.
- 오답인 문제의 기반 문구를 확인하면 원본 맥락을 파악하고 개념을 정확히 이해하는 데 도움이 됩니다.

> **Tip:** 틀린 문제의 결과를 클릭하여 원본 포스트에서 해당 내용을 다시 읽어보세요. 단순히 정답을 외우는 것보다 맥락과 함께 이해하는 것이 훨씬 효과적입니다.

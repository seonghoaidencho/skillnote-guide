# Skillnote 소개

## AI와 함께하는 스마트한 학습 복습 플랫폼

**Skillnote**는 개발자와 학습자를 위한 AI 기반 학습 관리 플랫폼입니다. 오늘 배운 내용을 기록하면, AI가 자동으로 복습 질문을 생성하고 간격 반복 학습(Spaced Repetition)을 통해 장기 기억으로 전환할 수 있도록 도와줍니다.

---

## 핵심 개념

Skillnote의 학습 사이클은 다음과 같이 동작합니다:

```
학습 내용 작성 → AI 질문 자동 생성 → 매일 간격 반복 복습 → 장기 기억 강화
```

1. **작성하기**: TIL(Today I Learned) 게시물로 오늘 배운 내용을 기록합니다
2. **AI 질문 생성**: 작성한 내용을 AI가 분석하여 다양한 유형의 복습 질문을 자동으로 만듭니다
3. **매일 복습하기**: 간격 반복 학습 알고리즘에 따라 최적의 시점에 복습 질문이 제공됩니다
4. **AI 채점 및 피드백**: 답변을 AI가 채점하고 상세한 피드백을 제공합니다

---

## 주요 기능

### 학습 게시물 작성
강력한 리치 텍스트 에디터로 학습 내용을 기록하세요. 코드 블록, 수식, 표, 이미지, YouTube 영상 등 다양한 형식을 지원합니다.

### PDF 업로드
PDF 문서를 업로드하면 AI가 내용을 요약하여 자동으로 학습 게시물을 생성합니다.

### AI 기반 복습 시스템
O/X, 객관식, 빈칸 채우기, 단답형, 서술형 등 5가지 유형의 질문으로 다양한 방식으로 복습할 수 있습니다.

### 개발자 블로그
작성한 학습 게시물을 공개하여 다른 개발자들과 지식을 공유할 수 있습니다.

### Q&A 시스템
게시물별 질문에 대해 다른 사용자들과 토론하고 답변을 나눌 수 있습니다.

### 다국어 지원
한국어, 영어, 중국어를 지원하며 언제든지 언어를 전환할 수 있습니다.
